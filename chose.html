# 快速开始

首先，请根据您的具体应用场景，选择您使用的工具：

如果您需要一套web服务，将您的数据发布为空间服务（比如空间查询、属性查询、矢量瓦片等），您可以使用 giscat server

如果您需要做一些更加自定义的gis场景，可以使用giscat tools提供的基础对象及其序列化、分析工具、瓦片生成工具等
