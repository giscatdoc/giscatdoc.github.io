# layer 图层

# 条件查询

# url

/Layer/{id}/Query

# 参数

参数名 类型 必填 默认值 说明
id string 是 -- 图层id
properties string 否 null 返回的要素包含哪些属性,以半角逗号分隔,若为null则表示不包含任何属性
expression string 否 null 条件表达式,若为null则表示不输入过滤条件
bindParams string 否 null 条件表达式绑定参数,若为null则表示不输入过滤条件，若expression为null则此参数不生效
f string 否 json 返回值格式，支持json和pbf，json返回geojson格式的FeatureCollection数据，pbf返回ProtoFeature格式的FeatureCollection数据
z byte 否 -1 查询的层级，为负则表示忽视图层rule中的zoom限制并取第0个rule

# 返回值

geojson or ProtoFeature 格式的 FeatureCollection 返回值详情点击这里

# 最邻近查询

# url

/Layer/{id}/Nearest

# 参数

参数名 类型 必填 默认值 说明
id string 是 -- 图层id
x double 是 -- 输入点的经度
y double 是 -- 输入点的纬度
n int 是 -- 最多返回多少条数据
properties string 否 null 返回的要素包含哪些属性,以半角逗号分隔,若为null则表示不包含任何属性
expression string 否 null 条件表达式,若为null则表示不输入过滤条件
bindParams string 否 null 条件表达式绑定参数,若为null则表示不输入过滤条件，若expression为null则此参数不生效
f string 否 json 返回值格式，支持json和pbf，json返回geojson格式的FeatureCollection数据，pbf返回ProtoFeature格式的FeatureCollection数据
z byte 否 -1 查询的层级，为负则表示忽视图层rule中的zoom限制并取第0个rule
