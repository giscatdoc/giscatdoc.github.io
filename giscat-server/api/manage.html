# 管理类接口

由于giscat-server是不带有鉴权功能的，所以绝对不要把管理类接口从网关转发出去，否则会有极大的安全问题。

例如，如果使用nginx做网关，可以考虑直接把管理类接口直接返回错误信息：

    location ^~ /giscat-server/Manage {
        default_type text/plain;
        return 403 "错误:安全起见manage请求已屏蔽";
    }

# 重新加载配置内容

# url

/Manage/ReloadConfig/{sort}

# 参数

参数名 类型 必填 默认值 说明
sort int 是 -- 从第几步开始重新加载

配置文件的加载分为很多个步骤，每个步骤都有一个sort编号，此接口被调用时，大于等于输入sort的步骤都将被执行，所有步骤如下：

sort 重新加载内容
0 插件
100 数据连接
200 数据集
300 LayerDataRule
301 图层
302 图层与地图的关系
400 地图
700+ 各类服务

# 返回值

success或抛出异常

# url

/Manage/ClearVectorTile/{id}

# 参数

参数名 类型 必填 默认值 说明
id string 是 -- 地图id

# 返回值

success或抛出异常
